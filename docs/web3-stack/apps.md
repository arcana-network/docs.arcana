# Supported App Types

{{config.extra.arcana.company_name}} SDKs can be used by different types of Web3 apps for enabling [[concept-social-login|social login]] and allowing users to sign blockchain transactions.

=== "App Frameworks"

    * Web
        - Vanilla HTML/CSS/JS
        - Vue
        - React/Next.js

    * Mobile
        - Flutter
        - React-Native

    * Gaming
        - Unity

=== "Wallet Connectors"

    * Wagmi
    * RainbowKit
    * WalletConnect
    * Web3-React
