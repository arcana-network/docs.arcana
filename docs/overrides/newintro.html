Developer Resources

Register & Configure

Use Arcana Developer Dashboard to register the app, configure authentication providers and blockchain signing experience. Get clientId for SDK integration.

Integrate Auth SDK

Choose your Web3 App type and follow the instructions to install, integrate with one or more required Arcana SDKs.

Plug-and-Play UX

Onboard users easily by simply configuring authentication providers and using the built-in login UX in the app. Select Web3 app type and follow instructions.

Custom UX

Onboard users by building custom login UX in the app and wiring it up with Arcana Auth functions, after configuring authentication providers.

Web3 Wallet Ops

Authenticated users can access the wallet UI. Developers can programmatically wire Web3 wallet operations supported by the Arcana wallet.

Deploy App

Deploy Web3 apps on Arcana Testnet, validate them and then deploy them on the Arcana Mainnet.

User Guides

See Arcana Developer Dashboard User Guide, Arcana Wallet User Guide and Multi-Factor Auth User Guide to enable enhanced wallet security.

How-to Guides

See step-by-step instructions on how to register the app, configure authentication providers, integrate with the Auth SDK, onboard users, deploy apps and more.
