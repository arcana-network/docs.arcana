=== "npm"

    **Wagmi 2.0**

    ```bash
    npm install --save @arcana/auth-wagmi@{{config.extra.arcana.latest_auth_wagmi_sdk_version}} @arcana/auth@{{config.extra.arcana.latest_version}}
    ```

    **Wagmi 1.0**

    ```bash
    npm install --save @arcana/auth-wagmi@2.0.0 @arcana/auth
    ```
